About ETSystem, Expense Tracking System

This browser based project lets you record expenses, manage wallets and view simple analytics. It is designed as a teaching example for HTML, CSS, JavaScript, Bootstrap and localStorage.

What users can do
  • Add expenses with category, date and wallet
  • Track multiple wallets like HBL, UBL and cash
  • See totals on dashboard and summary
What you can teach
  • Form handling and input validation
  • DOM updates with JavaScript
  • Saving data using localStorage
Student exercises
  • Add new categories and filters
  • Change theme or gradient colors
  • Extend summary with more reports
